# Advanced Secure Login Authentication System using PHP and MySQL

This project is a secure login authentication system developed using PHP, MySQL, Bootstrap, and XAMPP. The system provides user registration, login authentication, session handling, logout functionality, and password reset features with proper database connectivity and password security.

## Technologies Used

- PHP
- MySQL
- Bootstrap 5
- HTML
- CSS
- XAMPP

## Features

- User Registration
- Secure Login Authentication
- Password Hashing
- Session Management
- Forgot Password Functionality
- Logout System
- Responsive User Interface
- MySQL Database Connectivity

## Project Files

-  config.php → Database connection setup
-  register.php → User registration page
-  login.php → User login page
-  dashboard.php → Dashboard after successful login
-  logout.php → Logout functionality
-  forgot_password.php → Password reset page
-  index.php → Main entry page
